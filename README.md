## ✨ Key Features

- **💎 Glassmorphism Design:** A modern, premium aesthetic using backdrop filters and vibrant gradients.
- **📱 Fully Responsive:** Optimized for all devices, including a custom-built mobile navigation menu.
- **🎭 Smooth Micro-Animations:** Leveraging `framer-motion` for engaging entry transitions and hover effects.
- **📊 Business Impact Focused:** Structured to highlight quantifiable technical results and "Strategic Assets."
- **⚡ Performance Optimized:** Built on Next.js 15 for lightning-fast loading and SEO excellence.
- **📄 Dynamic Case Studies:** Integration for deep-dive project analysis (SQL, Python, Power BI).

## 🛠️ Technical Stack

- **Frontend:** Next.js 15 (App Router), React 19
- **Styling:** Tailwind CSS 4.0 (Modern CSS-first approach)
- **Animations:** Framer Motion 12
- **Icons:** Lucide React
- **Language:** TypeScript (Type-safe codebase)

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm / yarn / pnpm

### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/[your-username]/data-analyst-portfolio.git
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Run the development server:**

   ```bash
   npm run dev
   ```
